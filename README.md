# python-socket-server-from-scratch-
A minimal HTTP web server built from scratch using Python sockets.


This project demonstrates how HTTP works internally without using frameworks like Flask or Django.

Features:
- TCP socket server
- Manual HTTP request parsing
- Custom HTTP headers
- Routing support (/ , /about , /hello?name=...)
- Status codes (200, 400, 404)
- Query parameter parsing

This project is for learning backend fundamentals and understanding how web frameworks work internally.
